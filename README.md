## What is it?
`openmap-proxy` is the proxy layer for OpenPokemap

## Installation
1. Set your go path
2. Run `go get github.com/gorilla/websocket`
3. Run `go get github.com/Sirupsen/logrus`
4. Run `go run *.go`

And a local instance of the server should be running on your computer.
